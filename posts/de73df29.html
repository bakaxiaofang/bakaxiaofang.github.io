注册github账户并在本地git配置多个github账户

Created2024-01-31|Updated2024-02-01|GitHub
|Post Views:

一、注册GitHub账户

1.注册GitHub账户

GITHUB官网链接
Email address “电子邮件”
Sign up for GitHub “注册GitHub”
在 Email address 内输入邮箱 点击 Sign up for GitHub注册GitHub.
点击 Continue”继续” 输入密码与名称.
验证码不加载的话翻个墙.WattToolkit有github加速
点击 Create account”创建账户”
邮箱会获得启动代码 全选数字Ctrl+C复制,
回到github注册页面Ctrl+V粘贴.
后面选择Continur for free”继续免费”
Free “自由的” 免费.
Team “团队” 每月4美元.

二、在本地配置多个GitHub账户

!注:这里使用的是Windows系统
使用SSH连接到GitHub

1.生成ssh-key

详情请参阅生成ssh密钥
GitHub
2022年3月15日后不再支持DSA密钥.
2021年11月2日之前带有valid_after的RSA密钥可以继续使用任何签名算法,但在该日期之后生成的RSA密钥必须使用SHA-2签名算法.

C:\Users\FangB.ssh
输出
Enter file in which to save the key (C:\Users\FangB/.ssh/id_ed25519):
“输入要保存密钥的文件”
如果只用单个GitHub账户,可以按Enter”回车”键接受默认文件位置与默认名称,并跳到4. git config配置
默认路径:C:\Users\FangB.ssh “系统盘:\用户\用户名.shh”
如果多个的话,需要路径+用户名来区分不同账户.
例如: C:\Users\FangB\.ssh\id_ed25519_bakaxiaofang
接着输出
这里的密码填不填都行,具体参阅 [使用SSH密钥密码](https://docs.github.com/zh/authentication/connecting-to-github-with-ssh/working-with-ssh-key-passphrases)
Enter passphrase (empty for no passphrase):
“输入密码” “(空表示没有密码)”
Enter same passphrase again:
“再次输出相同的密码”
修改密码

打开~/.ssh/config文件,若没有则新建一个.

vim配置与下载请参阅 (-w-我还没写)
没有vim可以直接用记事本编辑

4. git config配置

如果之前登录过一个账户,请清空默认的全局 user.name 和 user.email并重启一次系统,不然会出错.

重启一下

GitHub添加SHH密钥并在本地测试SSH连接

点开头像,再打开Settings”设置”页面
点击SSH and GPG keys”SSH 和 GPG 密钥”
再点击New SSH key”新的 SSH 密钥”
Title”标题”随便
Key”密钥”
通过cat指令获得

将输出内容全选Ctrl+C复制,回到Add new SSH Key页面Ctrl+V粘贴
Add SHH key”添加 SSH 密钥”

只要输出为Hi GitHub用户名! You’ve successfully authenticated, but GitHub does not provide shell access.就是ssh连接成功.

hexo上传测试

例如hexo github.io
单账户: git@github.com:bakaxiaofang/bakaxiaofang.github.io.git
多账户: (根据ssh config修改)git@github.com-bakaxiaofang:bakaxiaofang/bakaxiaofang.github.io.git
在blog.deploy_git.git\config文件最后面添加
[user]
name = GitHub用户名
email = GitHub邮箱
