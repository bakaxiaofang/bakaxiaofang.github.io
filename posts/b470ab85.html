一、下载win-acme

1.客户端选择

Let’s Encrypt 官网
本来官网是推荐用Certbot客户端的,但是这个的Windows版本没法安装插件certbot-dns-cloudflare
ACME客户端选择详情请参阅ACME客户端
DNS插件参阅certbot DNS插件
详情参阅GitHub上提交的issues
issues9589
issues8048
接着还想着使用acme.sh,但是需要安装Cygwin,我嫌麻烦就算了’W’
win-acme 官网
win-acme GitHub
certbot 官网
certbot GitHub
acme.sh GitHub

2.下载win-acme和dns cloudflare插件

在win-acme 官网或GitHub 页面下载
!注: 选择pluggable版本,trimmed 较小版本不支持插件.
我在GitHub页面下载的版本 win-acme.v2.2.7.1612.x64.pluggable.zip
将下载的压缩包随便放在一个文件夹内并解压.
解压的文件有
win-acme.v2.2.7.1612.x64.pluggable
-Scripts
-public_suffix_list.dat
-settings_default.json
-version.txt
-wacs.exe
-Web_Config.xml
插件下载页面
插件问题具体请参阅win-acme Cloudflare
插件下载地址
2.2.7cloudflare直连下载
将下载的插件解压到pluggable版本的win-acme根目录里.
