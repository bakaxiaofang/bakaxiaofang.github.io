关于开启HDR截图谷歌浏览器曝光

GitHub issues

chrome://flags/
搜索Force color profile
修改为sRGB 但这样视频就没有HDR了，不推荐。

也可以用微软自带的截图Win + Shift + S，但这样截图也有轻微曝光。

没有太好的解决办法，个人建议熟练使用Win + Alt + B快捷键关闭HDR
