创建Cloudflare API令牌

Created2024-02-02|Updated2024-03-11|建站Cloudflare
|Post Views:

一、Cloudflare创建API令牌

1.找到API令牌页面

具体参阅宣布API Token普遍可用

如果你的网页不是中文,点击右上角的English,更换到简体中文.

要创建你的第一个API Token”令牌”,需要转到用户个人资料的API Tokens部分

在Cloudflare主页面点击右上角的小人,再点击我的个人资料.

将鼠标移到页面左边,点击API令牌.

或者直接链接跳转到API Token

2.创建令牌

点击创建令牌

​​

使用编辑区域DNS模板

​​

编辑权限已被预先选择

还需要读取权限

在权限下点击添加更多

选择 区域 DNS 读取

​​

区域资源下选择自己需要API令牌的域名

依次点击 继续以显示摘要、创建令牌

此时会显示用户 API 令牌

​!千万不要刷新​ 当然刷新了就删除这个令牌重新创建吧－w－

一定要Copy”复制” 并存放到记事本上

​!注意:千万不要泄露自己的API令牌​
