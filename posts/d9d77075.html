Hyprland(Wayland)火焰截图软件的安装与配置

发表于2024-11-07|更新于2024-11-07
|阅读量:

你还在为Linux截图麻烦而烦恼吗（并非麻烦）

想使用软件而非快捷键脚本截图吗（并非想）

这里由衷的推荐火焰截图官网地址 GitHub

我这里只讲Archlinux下的包，其它的发行版请自行寻找官方包

🔥🖼 flameshot 火焰截图 :给予你火一般的轻松配置

下载与安装

注意如果你要良好的使用体验，请使用最新版。

官方包正式版还不能很好的支持Hyprland。

类似这样的问题issues_3737

Archlinux AUR包

更多格式

你需要下载对于格式支持的前置库，才能使用呢。

AVIF
sudo pacman -S libavif

WEBP
sudo pacman -S libwebp

配置

我没有作任何其它的配置，就可以使用了，如果你遇到了问题请先尝试使用AUR最新版而非Arch官方包，其次再去看issues。

请自行打开软件作符合你操作的配置。

PicGo

AUR包

picgo-appimage

picgo-git

piclist-bin

维护者孤儿包佬的包好久没更新了，但还可以用。

维护者zxp19821005佬的包需要下载很大的文件，如果网络不好或者有限制，可能会下载失败。

请自行选择。

配置

懒的写直接跳转之前的文章Cloudflare R2储存图床+PicGo上传图片

与输入法有关的问题

你需要有qt包

其次虽然KDE勿加 GTK_IM_MODULE 和 QT_IM_MODULE.

但是Hyprland需要添加

env = GTK_IM_MODULE,fcitx

env = QT_IM_MODULE,fcitxm
